# SMENA — Система планирования рабочих смен

> **Курсовая работа** по дисциплине «Фронтенд разработка»  
> Тема: *«Клиентская часть системы планирования и визуального отображения рабочих смен персонала»*

---

## 📋 О проекте

**SMENA** — SPA-приложение для управления расписанием рабочих смен. Позволяет визуально распределять смены по сотрудникам, контролировать их доступность и анализировать отработанные часы.

### Стек технологий

| Технология | Назначение |
|---|---|
| Vanilla JavaScript (ES Modules) | Бизнес-логика и UI |
| Vite 8 | Сборщик и dev-сервер |
| CSS Custom Properties | Дизайн-система (dark theme) |
| GitHub Actions | CI/CD → GitHub Pages |

## ✨ Функциональность

### Календарь
- **📅 Недельный вид** — временна́я сетка 00:00–24:00 с абсолютным позиционированием смен
- **📆 Месячный вид** — стандартная сетка с чипами смен в ячейках
- **🌙 Ночные смены** — автоматически разрезаются по полуночи (первая часть → до 24:00, продолжение → с 00:00 следующего дня)
- **Перекрывающиеся смены** — корректная обработка кликов: вложенная (короткая) смена всегда получает приоритет над длинной

### Управление сменами
- **✏️ CRUD смен** — создание (клик по ячейке / кнопка), редактирование и удаление
- **👥 Мультиселект сотрудников** — несколько сотрудников на одну смену
- **🚫 Проверка статуса** — нельзя назначить смену сотруднику в отпуске или на больничном; такие сотрудники выделяются в форме цветными бейджами
- **🔔 Toast-уведомления** — при сохранении, обновлении и удалении
- **🪟 Кастомные диалоги** — все ошибки и подтверждения удаления показываются в красивых модальных окнах (без стандартного `alert`)

### Сотрудники
- **👤 Справочник персонала** — боковая панель с аватарами и индикатором статуса
- **➕ CRUD сотрудников** — добавление, редактирование (имя, должность, цвет, статус), удаление
- **🏷️ Статусы** — «Работает» / «Отпуск» / «Больничный» с цветовой индикацией

### Фильтры и навигация
- **🔍 Фильтры** — по сотруднику и типу смены (дневная / вечерняя / ночная)
- **📐 Сворачиваемый сайдбар** — кнопка «‹» на десктопе, кнопка «☰» на мобильных

### Статистика
- **📊 Клик на «SMENA»** → модальное окно со статистикой за текущий отображаемый месяц:
  - Количество сотрудников, которые работали
  - Суммарные **дневные часы** (07:00–22:00)
  - Суммарные **ночные часы** (22:00–07:00)
  - Детальная таблица по каждому сотруднику

### Адаптивность
- **📱 Мобильные устройства** (≤ 900 px) — сайдбар выезжает поверх, навигация компактная
- **📱 Маленькие экраны** (≤ 600 px) — модальные окна снизу (bottom sheet), компактный календарь

## 🚀 Запуск проекта

### Требования

- [Node.js](https://nodejs.org/) версии **18+**

### 1. Клонирование репозитория

```bash
git clone https://github.com/<твой-username>/Kursachella.git
cd Kursachella
```

### 2. Установка зависимостей

```bash
npm install
```

> ⚠️ **Windows + PowerShell**: если получаешь ошибку `UnauthorizedAccess`, выполни в PowerShell от администратора:
> ```powershell
> Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
> ```
> После чего повтори `npm install`.

### 3. Запуск dev-сервера

```bash
# Windows (PowerShell — если выдаёт ошибку скриптов)
cmd /c "npm run dev"

# macOS / Linux / cmd.exe
npm run dev
```

Открой браузер и перейди по адресу: **http://localhost:5173/**

### 4. Продакшн-сборка (опционально)

```bash
npm run build
```

Готовые файлы появятся в папке `dist/`.

## 🌐 Деплой на GitHub Pages

Деплой настроен автоматически через GitHub Actions (`.github/workflows/deploy.yml`).

**Шаги для подключения:**

1. Создай репозиторий на GitHub с именем `Kursachella`
2. Запушь проект:
   ```bash
   git init
   git add .
   git commit -m "init: SMENA"
   git remote add origin https://github.com/<username>/Kursachella.git
   git push -u origin main
   ```
3. В настройках репозитория → **Settings → Pages → Source** выбери **GitHub Actions**
4. После пуша GitHub Actions автоматически соберёт и задеплоит проект

Приложение будет доступно по адресу:  
`https://<твой-username>.github.io/Kursachella/`

## 📖 Как пользоваться

| Действие | Как сделать |
|---|---|
| Создать смену | Кликнуть на пустую ячейку дня **или** кнопку «+ Добавить смену» |
| Редактировать смену | Кликнуть на цветную карточку смены |
| Удалить смену | Открыть редактирование → кнопка «🗑 Удалить» |
| Фильтровать по сотруднику | Кликнуть на имя в левой панели **или** выбрать в фильтре |
| Сбросить фильтры | Кнопка «Сбросить» в левой панели |
| Переключить вид | Кнопки «Неделя» / «Месяц» в правом верхнем углу |
| Навигация по неделям/месяцам | Кнопки «‹» и «›» рядом с датой |
| Статистика за месяц | Кликнуть на надпись **SMENA** в левом верхнем углу |
| Открыть боковую панель (моб.) | Кнопка «☰» в левом верхнем углу |
| Свернуть боковую панель (ПК) | Кнопка «‹» в шапке боковой панели |
